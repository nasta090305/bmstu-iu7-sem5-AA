# Лабораторные работы по курсу "Анализ алгоритмов" МГТУ ИУ7. 5 семестр, осень 2023.

## Преподаватели

- Волкова Л.Л.

## Цель курса

Курс направлен на то, чтобы научиться оценивать эффективность и корректность алгоритмов, разбирать их временную и пространственную сложность и понимать, какие методы подходят для решения разных типов задач. Курс включает в себя освоение практической реализации алгоритмов, сравнение различных подходов, анализ их производительности и проверку корректности работы на тестах.
